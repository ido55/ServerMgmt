How to Add a Server

Servers are added automatically once the management agent is installed and connected.

Quick Steps:
  1. Install requests and psutil on the target server.
  2. Configure server_url and api_key in the agent's config.json.
  3. Run the agent (core.py) or register it as a service.

Agent Installation Instructions

1. Prerequisites

Python 3.x is required on the target server.

pip install requests psutil

2. Configuration

Create a config.json file in the following location:

  • Linux: /etc/agent/config.json
  • Windows: C:\ProgramData\Agent\config.json

Example Content:

{
  "server_url": "{{ request.url_root.rstrip('/') }}",
  "api_key": "default_secret_key"
}

3. Run Agent

Copy the agent/ folder to the target server and run:

python core.py

Or use register_service.py to install as a background service.
